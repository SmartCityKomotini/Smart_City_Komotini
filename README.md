# Smart_City_Komotini
Project Smart City
Γειά σας! Είμαστε η ομάδα Smart City of Komotini από την Κομοτηνή. 
Είμαστε μια ομάδα παιδιών δημοτικού σχολείου και έχουμε παρατηρήσει στην πόλη μας ότι πολλά σημεία της από τη μια δεν είναι ασφαλή και από την άλλη δεν σέβονται το περιβάλλον. 
Για το λόγο αυτό θα θέλαμε να προτείνουμε ένα έργο το οποίο θα δημιουργήσει μια πόλη ασφαλή αλλά και πράσινη η οποία θα προσφέρει σε όλους τους κατοίκους και τους επισκέπτες της ένα περιβάλλον στο οποία θα θέλουν να ζουν. 
Θα δημιουργήσουμε μια μακέτα που θα προσομοιάζει μια συνοικία της πόλης μας και θα εφαρμόσουμε σε αυτήν τεχνολογίες οι οποίες στο σύνολό τους θα εξασφαλίζουν μια καλύτερη καθημερινότητα για όλους.  Η έξυπνη πόλη μας θα έχει:
•	Φωτοβολταικά και ανεμογεννήτριες 
•	Αυτόματα φώτα
•	Αυτόματο πότισμα στα πάρκα
•	Αισθητήρες σε διαβάσεις και σιδηροδρομικές γραμμές
•	Φανάρια για ΑμΕΑ
•	Διαβάσεις για ΑμΕΑ
•	Αισθητήρες ελέγχου κυκλοφορίας 
•	Αισθητήρες για ασφαλή μετακίνηση παιδιών στα σχολεία 
•	Σημάνσεις για κινδύνους
•	Αισθητήρες ποιότητας αέρα 
•	Οθόνες πληροφόρησης

Υλικά για το project smart city of Komotini
•	Τέσσερεις πλακέτες Microbit 
•	Τροφοδοτικό
•	Ηλιακό πάνελ 
•	Αισθητήρες Microbit
•	Καλώδια και breadboard
•	Μακετόχαρτα 
•	Δύο σετ ρομποτικής της NEZA για την κατασκευή (για Microbit)
